The Schröder/Bernstein Theorem

Using our terminology the Schröder/Bernstein theorem states:
If two sets are size-bounded by each other then they are size-equal.
To show two, potentially uncountable, sets are size-equal, the proof constructs the necessary complete matching using the concept of a link chain. The first two sections of this chapter derive some properties of link chains. The proof itself follows in the the next two sections and is presented without explanatory examples. Try reading it that way, but if you have difficulty, the examples in the fourth section may help.

Proof Preliminary: Link Chains

Link chains are sequences of links from two matchings. They are not the same as the subset chains of Chapter 8. In this chapter and in many places they are simply called chains.
To define a chain we need two matchings R and Q between two sets A and B such that R covers A (i.e. e(R,A)=A) and Q covers B (i.e. e(Q,B)=B).
In this diagram I have made two copies of the same A and B to help differentiate between R and Q. The chains which interest us begin in B-e(R,B) like this
Define B* to be B-e(R,B) and a B*-chain to be a chain of links from R⋃Q like this
{c0,c1},{c1,c2},{c2,c3},…
where c0∊B*. In this section we will see that B*-chains exist exactly when B*≠empty and what their properties are.
P10.1
A potential B*-chain begins with a c0 which is in B* and continues with a link {c0,c1} which is in Q and whose other end c1 is in A. Moreover c0 is not in any other link of R⋃Q.
Proof
Can you prove this?
 

answer

By definition of a B*-chain c0∊B*. By definition of B*, c0 cannot be in any link of R. Since Q covers B, c0 is in a link {c0,c1} of Q. Since Q separates B, c0 cannot be in any other link of Q. Since Q is a relation between A and B, c1 must be in A.
P10.2
For any c∊B* a B*-chain exists and has these properties:
  1. For any odd i∊N:
    ci is in A and in exactly two links of R⋃Q, {ci-1,ci}∊Q and {ci,ci+1}∊R
  2. For any even i∊N:
    ci is in B and in exactly two links of R⋃Q, {ci-1,ci}∊R and {ci,ci+1}∊Q

Proof

Since I think of this chain as being constructed, link by link, from c0,c1, an inductive proof best fits my intuition. In the base case, i=1 and so P10.1 shows that {ci-1,ci}∊Q and c1∊A. Since R covers A, there is a {ci,ci+1}∊R. Since R and Q separate A, there can be no other links in R⋃Q which contain ci. Thus P10.2.1 is satisfied for i=1.

For the inductive step assume for an arbitrary natural number i>1, that
  1. For any odd j∊N with j<i:
    cj is in A and in exactly two links of R⋃Q, {cj-1,cj}∊Q and {cj,cj+1}∊R
  2. For any even j∊N with j<i:
    cj is in B and in exactly two links of R⋃Q, {cj-1,cj}∊R and {cj,cj+1}∊Q
Then for an odd natural number greater than 1:
i-1 is even and (taking j=i-1) satisfies the second inductive assumption). Therefore ci-1∊B and {ci-1,ci}∊Q so ci must be in A. Since R covers A, there is a ci+1∊B with {ci,ci+1}∊R. Since R and Q separate A, there can be no other links in R⋃Q which contain ci. Thus P10.2.1 is satisfied.
And for an even natural number greater than 1:
i-1 is odd and (taking j=i-1) satisfies the first inductive assumption). Therefore ci-1∊A and {ci-1,ci}∊R so ci must be in B. Since Q covers B, there is a ci+1∊A with {ci,ci+1}∊Q. Since R and Q separate B, there can be no other links in R⋃Q which contain ci. Thus P10.2.2 is satisfied.
Let's see what we can do with B*-chains knowing that P10.1 and P10.2 are satisfied.
P10.3
If for some c∊B*
C={c0,c1},{c1,c2},{c2,c3},…
is a B*-chain with c0=c, then for no i is there a {ci,x}∊R⋃Q that is not a link of C.
Proof
There is no such ci because any such
  • cannot be c0 (aka c) because of P10.1
  • cannot be ci with i∊N because of P10.2
We say a variable is in a B*-chain it there is a B*-chain with an link containing its value.
P10.4
Given any x∊A⋃B, either x is not an element of any link of some B*-chain or there exists exactly one B*-chain containing that link.
Proof
Let x be an element of A⋃B. We know that x either is, or is not, an element of a B*-chain so it is sufficient to show it cannot be the element of two distinct B*-chains. Let
C={c0,c1},{c1,c2},{c2,c3},…
be any B*-chain with some ci=x. If there exists a B*-chain distinct from C that also contains x, then (because the two chains must be different someplace) there must be a cj in C which is also in a link of that other B*-chain. By P10.3 that is impossible.
By P10.4, for every c∊B* there is exactly one B*-chain containing c. Let's call that chain Cc.

Proof of Schröder/Bernstein

Again we start with two matchings R and Q between sets A and B, where R covers A (i.e. e(R,A)=A) and Q covers B (i.e. e(Q,B)=B).
Here again is a representation of a single B*-chain
Let
Qc = {c0,c1},{c2,c3},{c4,c5},… Rc = {c1,c2},{c3,c4},{c5,c6},…
Define R' to be an alteration of R obtained by replacing all the Rc links in each Cc with Qc links. Altering R's links within each B*-chain this way does not ruin the fact that R separates A and B because each Cc is basically a silo of its own. P10.3 tells us no link of Cc has one end in e(Cc) and the other outside of e(Cc). P10.4 tells us e(Cc) shares no element with A⋃B-e(Cc).
Left to prove is that R' is a complete matching between A and B.
We know that R covers A and we can see from the definitions of Rc and Qc above that each Qc covers more than Rc. Therefore R' covers A.
Not only does each Qc cover more than Rc, the difference is an element of B*, namely c0. So when we replace the links of Rc with links of Qc in every Cc, we get that e(R')=e(R)⋃B*. Since B*=B-e(R,B) this means that e(R',B)=e(R,b)⋃(B-e(R,B)). Thus R' covers B too. In summary we have in R' a complete matching between A and B.
