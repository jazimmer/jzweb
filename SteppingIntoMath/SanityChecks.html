Sanity Checks

In this chapter we expand our theory of set sizing so that we can talk about smaller and larger sets. Then we check the theory for consistency with the rest of mathematics.

Size Definitions

Given sets A and B,
  • A is size-bounded by B exactly when there is a complete matching between A and a subset of B.
  • A is size-smaller than B exactly when A is size-bounded by B but not size-equal to B.
  • A is size-larger than B exactly when B is size-bounded by A but not size-equal to A.
These definitions describe how we are going to compare the sizes of any two sets. Before we can accept them we should check to see if they play nice with the rest of mathematics and whether we can use them to draw interesting conclusions about infinite sets. For those things we need more theory.
 
When S is a set of links or a sequence of links between sets A and B, we will write e(S) to represent the set of all elements of A or B that are in some link of S. Also, for simplicity,
  • e(S,A)=e(S)⋂A
  • e(S,B)=e(S)⋂B
 
P8.1
Any matching R between A and B is a complete matching between e(R,A) and e(R,B).
proof:
The definition of e(R,A) shows R covers e(R,A) and the definition of e(R,B) shows R covers e(R,B).

Theoretical Tool: Composing Matchings

Remembering that matchings are a kind of relation, let's look into a way that two relations can become one. The basic idea is shown here
The composition of two relations, R between A and B and Q between B and C, is a relation R∘Q between A and C defined this way
R∘Q = {{a,c}|there is a b∊B with {a,b}∊R ∧ {b,c}∊Q}
Said in words, R∘Q is a set consisting of links {a,c} formed from links {a,b}∊R and {b,c}∊Q whenever such a b exists.
Now notice
Q∘R = { {c,a} : there is a b∊B with {c,b}∊Q ∧ {b,a}∊R }
and so R∘Q= Q∘R.

P8.2
If R and Q are matchings, then R∘Q is a matching.
Proof
We know R∘Q is a relation. We need to show that it separates A and B.

We can show it separates A by showing any two links containing an element a of A must be the same link. To start assume {a,c}∊R∘Q and {a,c'}∊R∘Q.

From the definition of R∘Q we know there exist b,b'∊B with {a,b} and {a,b'} in R and {b,c} and {b',c'} in Q. Then b=b' (because R separates A) and and so c=c' (because Q separates B).

Thus R∘Q separates A.

Changing some variable names in the argument just given, we see that Q∘R separates C. Therefore since R∘Q=Q∘R, R∘Q separates C.

Therefore R∘Q is a matching.
 
P8.3
If R is a matching between sets A and B that covers A and Q is a matching between B and a set C that covers B. Then R∘Q is a matching between A and C that covers A.
Proof
Assume R is a matching between sets A and B that covers A and Q is a matching between B and C that covers B. Then by P8.2, R∘Q is a matching between A and C and all we have to prove is that R∘Q covers A.

Let a be an arbitrary element of A. Because R is a covers A there is a link {a,b}∊R. Given this b and because Q covers B, there is a link {b,c}∊Q. Because of b {a,c}∊R∘Q. Therefore R∘Q covers A.
 
P8.4
If R is a complete matching between sets A and B and Q is a complete matching between B and a set C. Then R∘Q is a complete matching between A and C.
Proof
Can you prove this??
 

answer

Assume R is a matching between sets A and B that covers A and B. Assume also that Q is a matching between B and C that covers B and C.

Since R covers A and Q covers B, P8.3 shows that R∘Q covers A. Since Q covers C and R covers B, P8.3 shows that Q∘R covers C. Therefore, since R∘Q=Q∘R, we know that R∘Q covers both A and C.

Theoretical Tool: Subset Chains

Given a set of sets S, a subset chain C is a subset of S such that if T and U are any elements of C, then either S⊆U or U⊆S. Subset chains are called chains in this chapter and many other places. They are different than the link chains found in Chapter 10.
For example let
S = { {1},{1,2}, {1,2,4}, {1,2,4,5},{1,3},{1,2,5} }
Then
C = { {1,2}, {1,2,4,5} }
is a chain of S.
A chain is maximal in case it is not a proper subset of another chain. In the above example, C is not maximal. Can you find all the maximal chains in S??
 

answer:

{ {1}, {1,3} } { {1}, {1,2}, {1,2,5 }, {1,2,4,5} } { {1},{1,2}, {1,2,4}, {1,2,4,5} }

Sanity Checking size-equal

If size-equal is to live up to its name, it must satisfy the properties mathematicians expect of equality. Translated to the language of size-equal these are:
  1. A set must be size-equal with itself. (reflexive property)
  2. When a set A is size-equal with a set B, then B must be size-equal with A. (symmetric property)
  3. When a set A is size-equal with a set B and a set B is size-equal with a set C, then A must be size-equal with C. (transitive property)
 
To show the reflexive property make a copy of the set A and then link each element with itself in the copy. The result is a complete matching between A and A.
 
To prove the symmetric property we need to show that a complete matching between A and B is also a complete matching between B and A. We already know this is true because the definition of complete matching is symmetric with respect to A and B.
 
The proof of the transitive property is found in P8.4.

Sanity Checking Size-Bounded

The sanity check for size-bounded asks whether it behaves like ≤. Translated to size-bounded the properties expected of ≤ are: for any two sets A and B
  1. a set A is size-bounded by itself (reflexive property)
  2. if A and B are size-bounded by each other, then they are size-equal. (antisymmetric property)
  3. Either A is size-bounded by B or B is size-bounded by A. (total-order property)
  4. If A is size-bounded by B and B is size-bounded by C, then A is size-bounded by C. (transitive property)
 
The reflexive property for size-bounded follows from the reflexive property for size-equal because the latter tells us that, among other things, there is a matching between A and any copy of A that covers A.
 
The antisymmetric property is proved by the Schröder/Bernstein theorem in Chapter 10.
 
To prove the total-order property let M be the set of all matchings between A and B. From P8.5 we know that there is a maximal chain C in M. Let m=⋃C. Then m contains only links between A and B. Therefore m is a relation between A and B.
That m is a matching can be proved this way:
Let x and y be any two links of ⋃C. Then (by definition of ⋃) there are matchings mx and my in C containing x and y respectively. (Of couse mx and my are not necessarily distinct.)

Since C is a chain either mx⊆my or my⊆mx. Since x and y are arbitrary we may as well say mx⊆my.
A common way to handle a similar situation in a proof is to say “Assume without loss of generality that …” and let the reader figure out why the proof isn't being damaged. So in this case: assume without loss of generality that mx⊆my.
Because mx⊆my, x and y are links in my which is a matching. Therefore (by P7.2) x and y have no element in common.

Since x and y are arbitrary links of m (aka ⋃C) that have no element in common, m is a matching between A and B (by P7.2 again).
We are interested in whether A covers B or vice-versa. (Both might be true but in this context we don't care.) There are three possibilities.
  1. m covers A, then A is size-bounded by B.
  2. m covers B, then B is size-bounded by A.
  3. neither 1) nor 2) is true; then by P8.1 there exists a∊A-e(m,A) and b∊B-e(m,B). This case is impossibile, a fact we can prove this way:
    Let
    C'=⋃C⋃{{a,b}}.
    Now notice that a and b have been chosen such that neither is an end point of a link in ⋃C. Therefore C' preserves ⋃C's property that no two links have an element in common. Therefore C' is a matching between A and B that is not in C (since it has a link no matching in C has). This contradicts the choice of C as a maximal chain in M.
Since only the first two cases are possible the total ordering property is proved.
 
The transitive property is proved in P8.3.

Revisiting Antisymmetry

The antisymmetric property tells us that if there are two matchings R and Q between sets A and B in which R covers A and Q covers B, then A and B are size-equal. All of Chapter 10 is devoted to proving this for both finite and infinite sets. When considering only finite sets, there is an easier proof.
Let A and B be finite and let there exist matchings, R and Q, between them that cover A and B respectively. Then
  1. |A|=|e(R,A)| because R covers A
  2. |e(R,A)|=|e(R,B)| because of P8.1
  3. Note |e(R,B)|≤|B| because e(R,B)⊆B.
  4. |A|≤|B| because of 1, 2 and 3
  5. Note |B|=|e(Q,B)| because Q covers B
  6. |e(Q,B)|=|e(Q,A)| because of P8.1
  7. Note |e(Q,A)|≤|A| because e(Q,A)⊆A.
  8. |B|≤|A| because of 5, 6 and 7
Therefore |A|=|B| because ≤ satisfies the anti-symmetry property.

Size-smaller than the Natural Numbers

Can an infinite set be size-smaller than N?
No, and here's a proof showing a set X cannot be both size-smaller than N and infinite.
Let X be an infinite set that is size-smaller than B. Then there exists a matching R between X and N that covers X.

Since X is infinite, we can count its elements without ever reaching an end to our counting.
This means that there is a matching Q between X and N that covers N and so (by anti-symmetry) X is size-equal to B.

Therefore there is no infinite set size-smaller than N.
