Mathematical Reasoning

Although mathematicians are experimenting with other forms of logic, mathematics itself is based on the belief that statements must be true or false but not both. A statement that is both true and false is a paradox. Paradoxes and mathematical reasoning do not get along with each other.
So around the beginning of the 20th century, mathematicians created rules that remove the possibility of paradoxes getting into mathematics. You saw an important one of these rules in the first chapter: a set may not directly or indirectly contain itself. The principle extends to more than sets. A statement like this
“This sentence is false.”
is forbidden. If you think about it, you will know it is paradoxical, but it is forbidden because it references itself.
Continuing with the same kind of thinking, a mathematical proof that a statement A is true or false may not directly or indirectly rely on the truth of A. However there is a way it can assume the falsity of A.

Proof by Contradiction

A proof by contradiction goes like this: To prove A use the possibility that ¬A (aka not A) is true as your starting point. If reasoning from that starting point leads to a paradox, then something is wrong. If your theory has no paradoxes and your reasoning is correct, that something must be the assumption that ¬A is true. Therefore A must be true.
By “leads to a paradox” is meant it is possible to prove something is false that is known to be true or something is true that is known to be false. In other words you are contradicting something you already know.
 
A common example of a proof by contradiction is a proof that there are infinitely many prime numbers.
A prime number is a positive integer that is divisible only by itself or 1.
How would you begin a proof by contradiction that there are infinitely many primes??
 

answer

Assume that there are only finitely many prime numbers.
 
If there are only finitely many prime numbers, we can list them in a sequence like this
p1,p2,p3,…,pn
Then let
Xp = p1×p2×p3×…×pn
Now consider Xp+1 which, like any integer greater than one, is either prime or divisible by a prime. Either possibility leads to a contradiction:
  1. If Xp+1 is prime then it is larger than all the pis and its existence contradicts our assumption that it is possible to list all the primes in a finite sequence.
  2. If Xp is not prime then it is divisible by a prime p. Our assumption says p is one of the pis and so p also divides Xp. Because p divides both Xp and Xp+1 there are numbers n and m such that n×p=Xp and m×p=Xp+1.

    Now Xp+1-Xp=1 and so substitituting for Xp and Xp+1,
    / m×p - n×p = 1
    or
    m - n = 1/p
    which is impossible because m and n are integers and 1/p is not (because p is a prime and primes are natural numbers greater than 1).
Therefore our assumption leads to a contradiction and so the number of prime numbers cannot be finite.

Mathematical Induction

Here is a sequence of assertions
A1, A2, A3, …
There is not much information here, only that we have infinitely many assertions each with its own natural number subscript. But if the Ais make the same claim about increasingly complicated data then mathematical induction may provide a technique for proving the claim is true for all Ais
Here is the proof process used by this technique.
  1. Prove A1. (the base case)
  2. Prove an arbitrary Ai (with i>1), assuming it is known that the Ajs that came before are true. (the inductive step)
I think of this as a proving machine: once the base case A1 is known to be true, the machine starts its work by proving A2 knowing A1, then A3 knowing A1 and A2, and so on. Of course the machine never finishes its job, but no matter what Ai you are interested in, the machine will get around to proving it in finite time.
What makes this different from simply picking an arbitrary Ai (with i>1) and proving it is true, is that the proof of Ai depends on knowing Aj is true for 1≤j<i.
As an example let's prove that
(n×(n+1))/2 = 1+2+3+…+n
for all n∊N. We start with the base case, which requires us to show that (1×(1+1))/2 is equal to the a sum that adds up the numbers in {1}. This is certainly true because the formula evaluates to 1, and the sum is also 1.
For the inductive step we need to prove that
(i×(i+1))/2 = 1+2+3…+i
for any i>1 under the assumption that
(j×(j+1))/2 = 1+2+3+…+j
for any natural number j that is less than i. This bit of knowledge is called the induction hypothesis.
For this proof what does the inducation hypothesis say about i-1??
1+2+3+…+(i-1) = ((i-1)×(i-1+1)) / 2
 
Using the induction hypothesis we get
1+2+3+…+i = i + ((i-1)×(i-1+1)) / 2 = 2×i/2 + ((i-1)×i) / 2 = (2×i+(i-1)×i) / 2 = ((2+i-1)×i) / 2 = (i×(i+1)) / 2
 
I like to incorporate the concept of induction with a technique that starts like a proof by contradiction. On this example this approach begins by letting Nope be the set of all i such that
(i×(i+1))/2 ≠ 1+2+3…+i
The desired result is true exactly when Nope=∅. A proof by contradiction begins by assuming Nope is not empty. Under that assumption we can pick a smallest i∊Nope. If this leads to a contradiction, we know Nope is empty.
Sometimes a proof by contradiction has to be broken into cases and it is necessary to prove each case leads to a contradiction. That is the situation here. We need to consider two cases: i=1 or i>1. Notice that this is the same division into cases we needed with a standard induction. The base case considered i=1 and the induction step proved the claim for i>1.
 
So the proof goes like this.
  1. This i cannot be 1 because
    (1×(1+1))/2 = 1
  2. This i cannot be greater than 1 because if it is then i-1∉Nope and so
    ( (i-1)×i) ) = 1+2+3+…+(i-1)
    This means
    1+2+3+…+i = i + ((i-1)×i) / 2 = 2×i/2 + ((i-1)×i) / 2 = (2×i+(i-1)×i) / 2 = ((2+i-1)×i) / 2 = (i×(i+1)) / 2
    showing i cannot be greater than 1.
  3. From 1. and 2. we see that Nope cannot contain a least integer which of course means Nope=∅ and so the result is true.

Propositional Logic

Metamathematics is the use of mathematics to investigate mathematics. Exactly which parts of mathematics you can use in metamathematics depends on which aspect of mathematics you are studying.
The Propositional Logic is a bit of mathematics that also qualifies as metamathematics. We can use it to make new inference rules for mathematics. The language of the Propositional Logic consists of atomic statements and compound statements. All statements can be referenced with variable names that begin with a capital letter and represent an assertion that something is true or false.
Although not done here, it is the statements of Propositional Logic are often referred to as propositions.
Atomic statements may be created by fiat, i.e.
let R be the statement, “It will rain today.”
 
let S be the statement, “Every natural number is prime.”
When atomic statements are created they are given a value of true (T) or false (F). This value does not come from the Propositional Logic, it is given to the Propositional Logic having been determined in some other way.
Compound statements are built up from other statements (from both atomic statements and compound statements). One way to do this is to negate a statement. So if A is T then ¬A (aka not A) is F and if A is F then ¬A is T.
Compound statements can also be built using one of the following three constructions:
  • Build A∧B from A and B -- think “and”
  • Build A∨B from A and B -- think “or”.
  • Build A⇒B from A and B -- think “implies”.
Here is a compound statement: A⇒(B∨¬C) and here is how it was built
C ¬C B∨¬C A⇒(B∨¬C)
This statement can be viewed as a theorem with premise A and conclusion B∨¬C or simply as an “if A then B∨¬C” statement with the same premise and conclusion.
An A⇒B statement can also be used to interpret a theorem. Consider this
If 2 divides n2, then 2 divides n
Clearly
  • the premise, “2 divides n2”, is represented with A, and
  • the conclusion, “2 divides n”, is represented with B.
Not quite so clear, because it cannot be expressed in the Propositional Logic, is that the theorem claims statement is true for an arbitrary integer n. In this case A⇒B represents a set of As and related Bs, one (A,B) pair for each n∊Z.
The Propositional Logic tells us how the truth values of these As and Bs must be related to each other. That relationship is expressed in a truth table. There is a truth table for each of the three kinds of compound statements. Here is the truth table for A⇒B
The first two lines of the table tell us that the above theorem is true (i.e. valid), if for every n with A true, B is also true and false (i.e. invalid) if, for any n with A true, B is false. The first line tells us what we have to establish to show the theorem is valid and the second line tells us what we have to establish to show the theorem is invalid.
The next two lines of the table tell us that if we misuse a valid theorem with a false premise, for example “7 divides 32”, then the conclusion is paradoxical, i.e. it can be both true and false.
Taken by itself the third line of this truth table seems incorrect, how can a true thing be concluded from a false thing?. It cannot, but there may be an (A,B) pair in which A is F and B is T. Consider this theorem
If a country is northenmost on the North American continent, then that country borders the United States.
This theorem can be applied to any country but the only country which makes the premise true is Canada. When A is “Mexico is northenmost on the North American continent” and B is “Mexico borders the United States”, A is F and B is T. The theorem isn't invalid. Rather the theorem is not inclusive enough to recognize all the countries which border the United States.
 
The other truth tables are easier to understand.
For example if A is true and B is false then the table on the left tells us that A∧B is false. Looking at the table on the right and thinking in reverse, suppose we know that A∨B is not true. There is only one row in the column for A∨B that has an F. The values of A and B in that row are both F. So the only way for A∨B to be false is for both A and B be false.

Using the Propositional Logic

As mentioned in the previous section the Propositional Logic is an example of mathematics that is sometimes used as metamathematics. In this section we look at an example of this.
Consider this table
The statement ¬B⇒¬A is called the contrapositive of A⇒B. Since the contrapositive is built up from ¬B and ¬A, columns for these values are also included in the table. The truth values for ¬B⇒¬A are determined by looking at the columns for ¬B and ¬A and applying the corresponding premise/conclusion pattern shown in the table defining A⇒B. This pattern is included in this composite table.
Because the columns for A⇒B and its contrapositive are the same, we know that any (A,B) pair that satisfies A⇒B will satisfy ¬B⇒¬A and vice versa. That tells us that to prove A⇒B, you can prove the contrapositive ¬B⇒¬A instead.
So proving the contrapositive is an inference rule that is derived from a metamathematical use of the Propositional Logic.
 
Here is an example of a theorem for which proving the contrapositive is valuable.
A = n2 is even
 
B = n is even
 
¬A = n is odd
 
¬B = n2 is odd
So the theorem says
For any n∊Z, if n2 is even, so is n.
A direct proof that A⇒B for any n∊Z isn't difficult if you know the Prime Number Theorem. If not, a proof of the contrapositive is easier. So, instead of proving the theorem, we will prove
if n is odd, then so is n2 ,
What makes this easier is that to draw conclusions about n2 from characteristics of n is easier than the other way around.
The trick to proving this contrapositive involves noticing that 2k+1 can represent any odd number as k varies over all integers. (This is just another way of saying that any odd number is one larger than some even number.)
So the proof goes like this;
Let n be odd, i.e. n=2k+1, for some k∊Z, then
n2 = (2k+1)(2k+1) = 4k2 + 4k + 1 = 4(k2+k) + 1
Since the last expression has the form of an odd number we know that n2 is odd and the result is proved.

Some elementary texts give lots of detail. Here I assume a reader who does not need to see definitions repeated again and again and who does not need to be told that to prove A⇒B you normally start with A and then reason your way to B.
