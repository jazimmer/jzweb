Sizing Sets

Are these two sets the same size?
One way to answer this is to say “No, because the one on the left has three elements and the one on the right has four.”
But when you try to count some sets, the counting never ends. We call those sets infinite. Finite sets of course are those that can be counted in finite time. To know a set is finite we must be able to show that counting will come to an end—although perhaps not in our lifetimes.
In this chapter we will look at a way to do endless counting. Later on we will consider the possibility that even endless counting is not enough to size some sets. But first we need examples of infinite sets to work with. Our number system provides plenty of examples.

Numbers Again

The real numbers R and two of its subsets, Z (the integers) and N (the natural numbers), are good subjects talking about things infinite because we have a lot of common understanding about their properties. In the previous topic we worked with intervals of real numbers. Here we will work with intervals of integers as well. To distinguish between different kinds of intervals we will use the following notation.
  • R[x,y] represents all the numbers between x and y including both x and y (x and y are numbers with x≤y)
  • Z[x,y] represents all the integers between x and y including both x and y (x and y are integers with x≤y)
  • R(x,y) represents all the numbers between x and y excluding both x and y (x and y are numbers with n<m)
  • Z(x,y) represents all the integers between x and y excluding both x and y (x and y are integers with x<y)
  • R(x,∞) represents the set of all numbers greater than x (x is any number)
  • Z(x,∞) represents the set of all integers greater than x (x is any integer)
  • R(-∞,y) represents the set of all numbers less than y (y is any number).
  • Z(-∞,y) represents the set of all integers less than y (y is any integer).
This notation makes it seem like ∞ is a number but it is just a symbol we call “infinity”. With R(x,∞) it means that no finite number we might put in its place will be big enough to describe the entire interval and with R(-∞,y) it means that no finite number we might put in its place will be small enough to describe the entire interval.
Extrapolating a bit from this notation can you find simple ways of representing these intervals Z[1,∞) and R(-∞,∞)??
 

answer:

Z[1,∞)=N and R(-∞,∞)=R.
 
By the end of this topic we will understand how R(0,1) is bigger than Z and the same size as R.

Counting

Imagine you are counting apples in a basket. One way to do it is to put labels, 1,2,3,…,n, on them. When the last label is n, that is the size of the set of apples in that basket.
This being a book about mathematics, our labels will be elements of N and, instead of pasting labels on set elements, we will match elements of N in a one-to-one way with the apples or whatever in the set to be counted. Of course the count starts with the label 1, and continues with 2, then 3, and so on.
Here is a representation of the kind of matching we use when determining the size of a set through counting.
The little squares represent unknown, but distinct, elements. The lines link these elements to numbers in Z[1,4]. Four numbers in Z[1,4] demonstrate there are four elements in S.
Lines are a good way to show links but sometimes they are too imprecise. For more precision we define a link between a set A and a set B as a two-element set with one element from A and the other from B. Each of these elements is an end of the link. A link between A and B is the same as a link between B and A because two element sets {x,y} and {y,x} are the same set.

Matchings

Our basic tool for investigating the relative sizes of sets is the matching. Here are some relevant definitions.
A set of links between a set A and another set B is called a relation. Since the link {a,b} is the same as the link {b,a}, relations are symmetric, i.e. a relation between A and B is also a relation between B and A.
A relation R
  • covers A in case every element of A is included in some link of R.
  • separates A in case every element of A is included in no more than one link of R.
Because of symmetry you can swap A and B in these definitions and understand how they apply to B.
Visually R separates both A and B when there no pair of links looks like either of these.
 
A relation is a matching in case it separates both A and B.
A matching is complete in case it covers both A and B. Here is an example of a set of links that is not even close to being a matching
because neither A or B is separated nor completely covered.
 
Here is a property of matchings we will need in later chapters.
P7.1
Any subset M' of a matching M between A and B is a matching between A and B.
Proof
Since M' is clearly a relation, all we need to know is that M' separates both A and B, i.e. no two links in M' have an end in common. They will not because if they did then there would be two links in M that have an end in common.
 
There is a subtlety here worth thinking about. The empty set ∅ is a subset of M and so P7.1 says it is a matching. This is ok because the condition that makes a relation a matching only talks about links in the relation. When there are no links, there is no condition to satisfy. The definition of a matching excludes relations whose links share an element and accepts all other relations.
 
P7.2
A relation R between A and B is a complete matching exactly when every element of A⋃B is in exactly one link of R.
Proof
All that is needed for this proof is to put the definitions of separates and covers together.
 
Let's return to thinking about the empty set because that can be tricky. Here is a question to practice on: can you say what has to be true for the empty set to be a complete matching?
 

answer

Since ∅ is always a matching, P7.2 tells us that all we need is for every element of A⋃B to be in exactly one link. So if either A or B has any elements, the empty set, having no links at all, cannot be a complete matching between them. But if both A and B are empty then the empty set is a complete matching because all the elements in A and B are in exactly one link of ∅.

A Theory of Set Sizes

The theory of set sizes we learned in elementary school has us counting the elements in a finite set until the counting stops with a number n. This number is called the cardinality of the set and, at least for a finite set S, it is denoted |S|.
The |x| notation has multiple meanings. It means absolute value when x is a number and cardinality when x is a set.
In this topic the concept of a complete matching replaces the concept of cardinality. We don't count elements of sets, unless they are finite. Instead we say two sets A and B are size-equal in case there is a complete matching between them. So when S is finite, it is size-equal with Z[1,|S|].
We say a set S is countable in case S is finite and has a complete matching with Z[1,|S|] or is infinite and has a complete matching with Z[1,∞). Countable sets always have a complete matching with a subset of N.
Sets for which no matching with a subset of the natural numbers exists are said to be uncountable. We will prove some sets uncountable after we have developed and sanity checked our theory for comparing the sizes of sets.

The Mystery of Proper Subsets

So N is a subset of the set of Z that seems to be much smaller because there are infinitely many integers of Z that are not included in N. Yet the two sets are size-equal. This requires some explanation.
An explanation has to do with the difference between comparing the sizes of two sets and comparing a set with a subset. The former generalizes counting to complete matchings and the later generalizes the concept of subset with that of proper subset.
There is a difference between counting that comes to an end and counting that does not. When we count the elements of a set and the counting comes to an end, we will have already counted the elements of any subset. This is what happens with finite sets.
However if both a set and a subset of that set are infinite, counting will never finish either one of them. This diagram shows such a case.
Both Z and N are counted. From 1 forward, every two arrows of counting Z becomes one arrow in counting N. Both sets are infinitely countable which means both can have complete matchings with an additional copy of N.
 
The basic ideas behind the theory for comparing set sizes presented here were established by German mathematician, Georg Cantor, who was active in the last part of the 19th century. However Cantor's imagination took him further than we will go. He defined the concept of a cardinal number that expresses the size of a set, finite or infinite. Because numbers don't make much sense without arithmetic, Cantor defined a kind of transfinite arithmetic that is consistent with the arithmetic we know.

Here we use Cantor's ideas without bothering with cardinal numbers or transfinite arithmetic. A theorem attributed to Ernst Schröder and Felix Bernstein helps make this possible.
