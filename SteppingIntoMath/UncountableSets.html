Uncountable Sets

In this chapter we show that R(0,1) is uncountable and is size-equal to R.
The question of whether there is a set size-larger than Z but size-smaller than R has not been answered, but it is generally believed that there is no such set.
The question of whether there are sets size-larger than R has been answered. There is a countable sequence of ever size-larger sets starting with R. You can study such things in a course on set theory.

To set up his diagonal argument Cantor made use of the fact that all the numbers in R(0,1) can be written in this pattern
0.d1d2d3…
with infinitely many fractional digits. Since a different sequence of fractional digits is needed for each ri, we add a second subscript to each dj so that for any i∊N
ri = 0.di,1di,2di,3…
Now R can be represented this way
We will create an element
x = 0.x1x2x3…
of R(0,1) which is not covered by R in a way that makes sure x is different from every ri by choosing xi to be any integer that is not di,i,0, or 9. (This leaves seven digits to choose from. Any of them will do.)
The reason this is a diagonal argument is that the xis are are chosen to be different from the di,is on this diagonal.
Because each choice for xi
  • excludes di,i, we know that x is different from ri.
  • is neither 0 nor 9, we know that x∊R(0,1).
Thus x is an element of R(0,1) that is different from any ri, a contradiction of the assumption that R is a complete matching between N and R(0,1).
Since there is a matching between N and R(0,1) that covers N but there is not any complete matching between N and R(0,1), N is size-smaller than R(0,1).

Theoretical Tool: Assembling Matchings

Sometimes the easiest way of creating a matching is to assemble it as the union of other matchings. The basic idea starts with two complete matchings m1 between A1 and B1 and m2 between A2 and B2 where {Ai,A2} partitions A and {Bi,B2} partitions B.
The idea is that m=m1⋃m2 will cover A and B and consist of links that have no common ends. It is not difficult to prove this, but pretty much the same proof works for a more general construction.
P9.1
Using any set I as an index set suppose we have sets A and B which have partitions
A = {Ai|i∊I} B = {Bi|i∊I}
where for each i∊I there is a complete matching
mi between Ai and Bi
then the relation m=⋃{mi|i∊I} is a complete matching between A and B.
Proof We need to know that m separates and covers both A and B. The reasoning is the same for both A and B. Here it is for A.
m separates A because
  • No element of any Ai is in two links from any single mi because all the mis are complete matchings.
  • No element of any Ai is in links from any two distinct mis because the Ais partition A.
m covers A because each element of A is in some Ai and mi covers Ai and ⋃{Ai|i∊I} = A.

Previous Y is Uncountable

You probably have an intuitive understanding that the Y defined in the previous section is uncountable. Here is a result that can be used to prove that fact.
An uncountable set cannot be partitioned into two countable sets.
A proof goes like this.
Assume otherwise, that there is an example of an uncountable set Z that has been partitioned into two countable sets {X,Y}. Then there are complete matchings mx (between N and X) and my (between N and Y). With mx and my we ought to be able to apply P9.1 in a way that shows that Z is countable.
 
When we know that Z is both countable and uncountable we have a contradiction proving that an uncountable Z cannot be partitioned into two countable sets.
