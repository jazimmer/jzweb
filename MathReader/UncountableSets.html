Uncountable Sets

In this chapter we show that R(0,1) is uncountable and is size-equal to R.
The question of whether there is a set size-larger than Z but size-smaller than R has not been answered, but it is generally believed that there is no such set. This belief is called the “Continuum Hypothesis”.
The question of whether there are sets size-larger than R has been answered. There is a countable sequence of ever size-larger sets starting with R. You can study such things in a course on set theory.

Abstract Tool: Partitions

A partition of a set S is a set of nonempty subsets P of S such that
  • ⋃P=S
  • every two distinct sets A and B in P are disjoint.

Without changing what is written, can you finish the sentence so that the definition is equivalent to the one above??
 

answer

A partition of a set S is a set of nonempty subsets P of S such that ⋃P=S and if any two sets in P have an element in common then they are the same set.
Note that “A⋂B≠∅ implies A=B” is the contrapositive of “A and B distinct implies A⋂B=∅”.

Abstract Tool: Assembling Matchings

Sometimes the easiest way of creating a matching is to assemble it as the union of other matchings. Of course these other matchings must have nothing in common.
Suppose we have a two sets A and B that are partitioned into an equal, finite number of subsets,
{A1,A2,…,An}
and
{B1,B2,…,Bn}
Further assume that there is a complete matching mi between Ai and Bi for all 1≤i≤n as in this diagram.
Let m=⋃{mi : i∊N & i≤n}.
P9.1: m as defined above is a complete matching.
Proof:
  • No two links in m have an element in common because
    • No element of any Ai (Bi) is in two links from any single mi (because all the mis are complete matchings).
    • No element of any Ai (Bi) is in links from any two mis (because the Ais (Bis) partition A (B)).
  • Any element of any A (B) is in some link of some mi because
    • Each mi is a complete matching that covers Ai and Bi.
    • The Ais partition A and the Bis partition B.
  • Therefore the links in m focus and cover both A and B which means m is a complete matching between A and B.

Abstract Tool: Algebraic Matchings

So far we have seen two kinds of diagrams for showing that two sets have a matching between them, using links or arrows. These diagrams are not very useful when the sets involved are uncountable. When working with uncountable sets, we turn to algebra. Consider this.
According to common practice, R(0,1) and R(0,2) are assumed to be disjoint copies.
Here we have one algebraic equation in x and y that defines this relation
{ {x,y} | x∊R(0,1)∧y∊R(0,2)∧2x=y }
The viewpoint from above the line shows that for every y∊R(0,2) there is exactly one x∊R(0,1). Thus the relation covers and separates R(0,2).
The viewpoint from below the line shows that for every x∊R(0,1) there is exactly one y∊R(0,2). Thus the relation covers and separates R(0,1).
You can see that algebraic matchings are meant to define complete matchings. It is the reader's job to verify that there is exactly one x for each relevant y and and vice versa.
